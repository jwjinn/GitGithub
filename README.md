## Git과 GitHub는 무엇이 다를까?

* Git: 분산 버전 관리 시스템
* GitHub: Git만 사용하게 될 경우에는 로컬에서만 버전 관리를 진행하게 됩니다. 인터넷을 통해서, 공유하기 위해서 사용하는 것이 GitHub입니다.

</br>

## GitHub의 주된 사용처

  1. 나의 결과물을 팀원들과 공유하기 위해서
  2. 나의 프로젝트를 다른 기기에서 쉽게 접속하기 위해서가 이 두가지가 가장 큰 목적일 겁니다.
